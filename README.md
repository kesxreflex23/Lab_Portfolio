# Lab Portfolio Project

A responsive personal portfolio website built as part of **SCSM2223 Lab Exercises**.  
This project demonstrates structured HTML5, external CSS architecture, Flexbox, CSS Grid, and responsive design principles.

---

## 📂 Project Structure
- `index.html` – Home page with card grid layout
- `about.html` – About me section with typography and sidebar layout
- `skills.html` – Skills page with lists styled using Flexbox
- `academic.html` – Academic record with accessible HTML table
- `projects.html` – Projects showcase with responsive image grid
- `contact.html` – Contact form styled with CSS pseudo-classes
- `style.css` – Single external stylesheet for all pages
- `images/` – Project screenshots and profile photo

---

## 🎨 Features
- **Semantic HTML5** structure (header, nav, main, section, footer)
- **Custom CSS variables** for colors, fonts, and spacing
- **Flexbox navigation** with sticky positioning
- **Responsive card grids** and sidebar layouts using CSS Grid
- **Mobile-first media queries** for 3 breakpoints:
  - Mobile: 320px+
  - Tablet: 600px+
  - Desktop: 1024px+
- **Form styling** with focus states, transitions, and pseudo-elements
- **Accessible design** (labels, alt text, focus indicators)
